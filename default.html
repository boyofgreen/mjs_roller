Manifoldjs Test Portal

Warning: This app provides unrestricted access to the WinRT surface and should be used for development purposes only. Think of the Children!

Q: What does this app do?

A: This Allows you to test your web app as a windows hosted web app, including access to the Windows Universal APIs

Q: What doesn't this app do?

A: It doesn't produce a true appx. Since it's a "test portal", it also doesn't honor your "mjs_access_whitelist" or scope. This means Windows API access is turned on for all of your URLs, and links to items outside of the scope will open in this test portal.
